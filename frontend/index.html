Platform Overview

PulmoScan AI is an enterprise-grade lung imaging intelligence platform built on transfer learning with EfficientNetB0. It classifies chest imaging into four categories: COVID-19, Normal, Pneumonia, and Tuberculosis.

  • 🔬 EfficientNetB0 transfer learning (ImageNet → medical fine-tune)
  • 🎯 Multi-class detection with confidence scoring
  • 🔥 Grad-CAM explainability heatmaps
  • 📱 Responsive clinical-grade UI
  • 🐳 Docker-ready deployment
  • 📈 Real-time dashboard & history tracking

Getting Started

# Install
pip install -r requirements.txt
pip install -r requirements-dev.txt

# Run tests
make test-fast        # quick unit + API tests
make test             # full suite incl. inference

# Train model
python ml_training/train.py --quick

# Start platform
python3 run.py

# CLI prediction
python3 scripts/predict.py scan.png

Original Research

Evolving from a GITAM University B.Tech project on COVID-19 detection using deep learning, PulmoScan AI expands the scope to a full lung imaging platform with modern architecture, explainable AI, and production deployment.

Authors: L. Naga Sai Sri Ravi Teja, S. Ritesh Dev, K. Bharath, T. Yashwanth Sai
Guide: Dr. Don S. Kumar, GITAM University
